Trajes Recomendados

Vestidos

Opte por vestidos elegantes, mas não tão formais quanto os das madrinhas. Cores claras ou pastéis são ótimas escolhas. Evite branco, que é reservado para a noiva.

Saltos/Sandálias

Saltos médios ou sandálias são ideais para um casamento vespertino.

Ternos

Um terno convencional é o ideal, mas o traje esporte fino também é uma ótima opção.

Sapatos/Tenis

Sapatos sociais ou mocassins são adequados. Tênis mais sociais também são bem-vindos.

Madrinhas e Padrinhos

Traje social completo. Não há exigências para cores de vestidos e nem de gravata.
